# Darts Leaderboard

## How to run locally

1. Clone the repo
2. Go to the repository
3. Create .env file

    - Add 2 environment variables

        ```
            REACT_APP_CLIENT_ID=<CLIENT_ID>
        ```

        ```
            REACT_APP_CLIENT_SECRET=<CLIENT_SECRET>
        ```

4. Install node modules

    ```bash
        npm start
    ```

5. Run the server

    ```bash
        npm start
    ```
